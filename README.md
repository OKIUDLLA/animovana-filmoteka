# 🎬 Animovaná filmotéka

Česká databáze animovaných filmů s českým dabingem — vhodných do kina i TV, nikoliv batolecí obsah.

## Obsah
- **index.html** — interaktivní databáze (154 filmů): hledání, filtry (studio, země, kategorie, dekáda, věk), řazení, detail filmu.
- **karta-lvi-kral.html** — ukázková detailní karta filmu (Lví král, 1994): děj, herci + český dabing, soundtrack, trailer, galerie, série a odkazy.
- **databaze.xlsx** — stejná data jako tabulka s filtry.

## Živá verze (GitHub Pages)
Po zapnutí v Settings → Pages (Branch: main /root) poběží na:
https://okiudlla.github.io/animovana-filmoteka/

## Poznámky
Hodnocení ČSFD / Kinobox / IMDb jsou orientační. Kurátorovaný výběr, ne kompletní katalog.
